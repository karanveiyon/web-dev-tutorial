HTML Tag Attributes Reference with Outputs

This document provides a reference for HTML tags, including their attributes, a one-line description for each attribute, and the rendered output for each tag.

1. Document Structure Tags

1.1 <!DOCTYPE html>

Declares the document type as HTML5.

<!DOCTYPE html>

[No visible output; defines the document as HTML5]

AttributeDescription
(No attributes)HTML5 doctype has no attributes.

1.2 <html>

<html lang="en">
    <body>Content</body>
</html>
Content
AttributeDescription
langSpecifies the language of the document (e.g., "en" for English).
dirDefines text direction (e.g., "ltr" or "rtl").
classAssigns one or more class names for CSS styling.
idAssigns a unique identifier for the element.

1.3 <head>

<head>
    <title>Sample</title>
</head>

[No visible output; contains metadata]

AttributeDescription
profileSpecifies a URI for metadata profile (rarely used).

1.4 <title>

<head>
    <title>My Page</title>
</head>

[Displayed in browser title bar: "My Page"]

AttributeDescription
(No specific attributes)Contains text for the document title.

1.5 <meta>

<head>
    <meta charset="UTF-8">
</head>

[No visible output; sets character encoding]

AttributeDescription
charsetDefines the character encoding (e.g., "UTF-8").
nameSpecifies metadata name (e.g., "viewport", "description").
contentProvides the value for the metadata name.
http-equivDefines HTTP header equivalent (e.g., "refresh").

8. Form Tags (Dropdown)

8.5 <select>

Creates a dropdown menu for selecting one or more options.

<form>
            <select name="dropdown">
                <option value="option1">Option 1</option>
                <option value="option2">Option 2</option>
            </select>
        </form>
Attribute Description
name Specifies the name of the dropdown for form submission.
multiple Allows multiple options to be selected.
size Sets the number of visible options in the dropdown.
required Indicates that a selection is mandatory.
disabled Disables the dropdown menu.
class Assigns one or more class names for CSS styling.
id Assigns a unique identifier for the element.
style Applies inline CSS styles.

8.6 <option>

Defines an option within a <select> dropdown menu.

<form>
            <select name="dropdown">
                <option value="option1">Option 1</option>
            </select>
        </form>
Attribute Description
value Specifies the data sent when the option is selected.
selected Pre-selects the option by default.
disabled Disables the specific option.
label Provides a shorter label for the option (rarely used).
class Assigns one or more class names for CSS styling.
id Assigns a unique identifier for the element.
style Applies inline CSS styles.
