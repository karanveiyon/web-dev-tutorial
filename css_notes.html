18. Transitions and Animations

div {
    width: 100px;
    height: 100px;
    background: lightblue;
    position: relative; /* required for left animation */
    transition: background 0.3s;
}
div:hover {
    background: rgb(255, 5, 46);
    animation: slide 1s forwards; /* apply animation on hover */
}

@keyframes slide {
    from { left: 0; }
    to { left: 100px; }
}
